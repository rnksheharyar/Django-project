# DJANGO

## Installation

### miniconda
 
```bash
conda create -n django python3.11 -y
conda activate django
pip install django

### Python environment

``` Bash
python -m venv .venv
source .venv/bin/activate # for linux/mac
.venv\Scripts\activate # for window
pip install django

## Create a Django project;

```bash
django-admin startproject # noor is the project name
cd noor
